# 学术研究智能助手 — Academic Research Agent

## 技能概述

帮助社会科学研究者处理研究流程中的三大核心痛点：

| 痛点 | 功能 | 输入 → 输出 |
|------|------|-------------|
| 文献阅读效率低 | 智能阅读 | PDF → 结构化笔记 |
| 访谈整理繁琐 | 智能编码 | 访谈文字 → 编码+主题+发现 |
| 报告撰写耗时长 | 段落生成 | 数据表格 → 分析段落 |

---

## 核心功能模块

### 1. 学术文献智能阅读 (literature-reading)

**触发条件：** 用户上传 PDF 或发送长文本并说"帮我分析这篇论文"

**处理流程：**
1. 提取论文关键信息
2. 生成结构化笔记
3. 提炼可借鉴之处

**输出格式：**
```markdown
## 论文分析笔记

**研究问题：** ...
**研究方法：** ...
**样本与数据：** ...
**核心发现：** ...
**可借鉴之处：** ...
```

### 2. 访谈资料智能编码 (interview-coding)

**触发条件：** 用户发送访谈文字并说"帮我编码"或"分析这段访谈"

**处理流程：**
1. 逐段识别关键概念 → 编码
2. 编码归类为主题
3. 提炼核心发现
4. 生成受访者画像
5. 输出研究结论初稿

**编码原则：**
- 概念命名：使用方括号标注，如【情感定位：自己的宝宝】
- 主题聚类：将相似编码归类为主题
- 引用追踪：每个编码标注原始引文

### 3. 数据分析描述生成 (data-description)

**触发条件：** 用户发送统计表格并说"帮我写描述性分析"

**输出示例：**
```
本研究样本共包含XXX名受访者，其中女性占XX%（N=XX），
平均年龄XX岁（SD=XX）。在职业分布方面，学生占比最高，
达XX%（N=XX），其次为...
```

### 4. 研究大纲辅助生成 (outline-generator)

**触发条件：** 用户发送研究主题并说"帮我设计论文框架"

**输出格式：**
```markdown
## 论文框架建议

### 一、引言
- 研究背景：...
- 研究问题：...

### 二、文献综述
- 核心概念界定：...
- 相关研究回顾：...

### 三、研究方法
- 研究设计：...
- 数据收集：...
- 分析方法：...

### 四、结论与讨论
- 主要发现：...
- 理论贡献：...
```

## Prompt 模板

### 访谈编码模板

```
你是一位专业的定性研究分析助手。请对以下访谈内容进行编码分析：

【访谈内容】
{interview_text}

请按以下步骤输出：

1. **逐段编码**
   - 提取每个关键表述的概念编码（用方括号标注）
   - 如："我会把它们抱在怀里" → 【情感功能：情绪调节】

2. **主题聚类**
   - 将相似编码归类为主题
   - 展示主题之间的关系网络

3. **核心发现**
   - 提炼3-5个核心发现
   - 每个发现标注支持证据（引用原文）

4. **受访者画像**
   - 基本信息（年龄、职业等）
   - 消费特征（偏好、频率、金额）
   - 情感特征（态度、互动方式）

5. **研究结论初稿**
   - 生成学术规范的研究结论段落
```

### 文献阅读模板

```
你是一位专业的学术文献分析助手。请对以下论文内容进行分析：

【论文内容】
{paper_content}

请按以下结构输出：

1. **研究问题**
   - 论文探讨的核心问题是什么？

2. **研究方法**
   - 定量/定性/混合？
   - 具体方法（问卷、访谈、实验等）

3. **样本与数据**
   - 样本量、数据来源、时间范围

4. **核心发现**
   - 主要结论与理论贡献

5. **可借鉴之处**
   - 对我自己研究的参考价值
```

## 使用示例

### 示例1：访谈编码

用户输入：
> 帮我编码这段访谈：
> "我觉得棉花娃娃更像是我自己生活中的一个'小宝宝'，我会给它们起名字，想象它们的性格和故事，甚至会和它们'说话'。每次看到它，我都会觉得它像是我的一个'小宝宝'，需要我去照顾和呵护。"

助手输出：
> **编码分析：**
> 
> - "我觉得棉花娃娃更像是我自己生活中的一个'小宝宝'" → 【情感定位：自己的宝宝】
> - "我会给它们起名字" → 【命名行为】
> - "想象它们的性格和故事" → 【拟人化建构】
> - "会和它们'说话'" → 【日常互动：对话】
> - "需要我去照顾和呵护" → 【照料行为】
> 
> **主题归类：** 【独立情感依附】【拟人化互动】

### 示例2：描述性分析

用户输入：
> 帮我写描述性分析：
> N=200，女性65%，平均年龄23.5岁，学生占70%

助手输出：
> 本研究样本共包含200名受访者，其中女性占65%（N=130），平均年龄23.5岁。在职业分布方面，学生占比最高，达70%（N=140），其次为企业职员（20%，N=40）和其他职业群体（10%，N=20）。

## 注意事项

1. **编码精度**：定性编码具有主观性，建议用户校验结果
2. **学术规范**：生成的文本需用户根据实际情况调整
3. **数据安全**：访谈数据涉及隐私，注意保密处理
4. **引用规范**：使用生成内容时需标注AI辅助
