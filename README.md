# 定性研究编码助手

> 自建的研究辅助工具，用于访谈资料的主题编码与概念提炼

## 背景

在做棉花娃娃消费者研究时，手头有二十多份访谈记录，纯人工编码太慢，于是写了这个工具来辅助处理。

主要用途：
- 访谈文本的初步编码（提取概念、标注主题）
- 多份访谈的横向对比
- 生成可供复核的结构化结果

## 功能

### 访谈编码
输入一段访谈文字，输出：
- 概念编码（方括号标注）
- 主题归类
- 核心发现提炼
- 受访者画像（可选）

### 示例

输入：
> "我会把它们抱在怀里，感觉心情就会好很多"

输出：
\\\
编码：【情感功能：情绪调节】【陪伴缓解焦虑】
主题：【情感慰藉】
发现：肢体接触可触发即时情绪改善
\\\

## 使用方式

本工具基于 OpenClaw 框架构建，可通过对话调用。

### 调用示例
`
用户：帮我分析这段访谈："[访谈文字]"

助手：
- 自动识别关键概念
- 归类到主题
- 输出编码结果
- 生成核心发现
`

## 技术说明

工具基于 OpenClaw 框架开发，使用结构化 Prompt 模板处理访谈文本。
编码逻辑参考扎根理论的开放编码流程，但做了简化适配。

生成的编码结果仍需人工复核，不替代研究者判断。

## 适用范围

适合社会科学定性研究中的访谈编码环节，特别是：
- 深度访谈记录
- 半结构化访谈
- 焦点小组记录

## 已知局限

- 复杂语境下的编码可能需要调整
- 跨访谈对比功能尚不完善
- 定量分析部分（CGSS等）不在此工具处理范围内
