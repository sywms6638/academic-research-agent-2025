# 🔬 学术研究智能助手 (Academic Research Agent)

> 基于 OpenClaw 框架开发的学术研究辅助智能体，帮助文科研究者提升文献阅读、访谈整理、报告撰写效率

## 📌 项目简介

**学术研究智能助手** 是一个运行在 OpenClaw 上的研究辅助智能体。它能够帮助社会科学研究者处理三大核心痛点：

| 痛点 | 解决方案 |
|------|---------|
| 📖 文献阅读效率低 | 自动提取研究问题、方法、结论，生成结构化笔记 |
| 📝 访谈整理繁琐 | 自动编码、主题提炼、受访者画像生成 |
| 📄 报告撰写耗时长 | 根据数据表格自动生成描述性分析段落 |

**参赛赛道：** 飞书 AI 校园挑战赛 · OpenClaw 赞助赛道  
**开发者背景：** 文科研究者（社会学/定量研究方向）

---

## 🎯 核心功能

### 1. 📖 学术文献智能阅读

**输入：** 学术论文 PDF  
**输出：** 结构化笔记

| 提取项 | 说明 |
|--------|------|
| 研究问题 | 论文探讨的核心问题 |
| 研究方法 | 定量/定性/混合方法 |
| 样本与数据 | 样本量、数据来源 |
| 核心发现 | 主要结论与贡献 |
| 可借鉴之处 | 对自己研究的参考价值 |

### 2. 📝 访谈资料智能编码

**输入：** 访谈文字记录  
**输出：** 编码结果 + 主题网络 + 受访者画像

```
原始文本 → 自动编码 → 主题聚类 → 核心发现提炼
```

**示例输出：**

```
受访者："我会把它们抱在怀里，感觉心情就会好很多"

→ 编码：【情感功能：情绪调节】【陪伴缓解焦虑】
→ 主题：【情感慰藉功能】
```

### 3. 📊 数据分析描述生成

**输入：** 描述性统计表格  
**输出：** 学术规范的描述性分析段落

> "本研究样本共包含XXX名受访者，其中女性占XX%，平均年龄XX岁..."

### 4. 📄 研究大纲辅助生成

**输入：** 研究主题  
**输出：** 论文框架建议

- 引言结构
- 文献综述框架
- 方法部分要点
- 结论部分建议

## 💡 设计理念

本项目的设计基于真实的文科研究场景：

1. **研究者视角**：开发者本身就是社会科学研究者，深谙研究流程中的痛点
2. **场景驱动**：功能设计来源于真实需求（文献阅读、访谈整理、报告撰写）
3. **学术规范**：输出内容符合学术论文写作规范
4. **可扩展性**：Prompt 模板可根据不同研究领域定制
